Web Development

Web development involves the creation and maintenance of websites.

WEBSITE/ WEBPAGE

A website is a collection of publicly accessible, interlinked Web pages that share a single domain name.

A webpage is a single page within a website. So, any individual part of a website that you access can be considered a webpage.
